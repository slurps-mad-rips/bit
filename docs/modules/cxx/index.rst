CXX Module
==========

.. module:: cxx
   :synopsis: Tasks related to C, C++, and Objective-C

The cxx module provides tasks and classes to use bit in a C, C++, or
Objective-C centric workflow. It tries to leave as much work as possible to
other tools, going so far as to use Mako templates for the configure task.
